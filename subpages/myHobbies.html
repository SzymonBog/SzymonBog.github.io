Modeling

To create a game you need code and assets. I learned how to write a code so what's left was assets. Of course I could buy assets but they are expensive, sometimes too expensive. So I decided that I have to learn how to create them(the small ones). To do that I downloaded 'blender' and started playing with it. It took some time but I managed to learn how to use it. I started by creating simple shapes and after some time I began to try to create more complex models like 'Scottish Claymore' with a fuller.
